Project Overview

SecureChat is a secure, real-time Android chat application designed for seamless text based communication. Users can send text messages, share photos, and post temporary stories, fostering a rich and interactive messaging experience.

Built using Firebase for authentication, real-time database, and cloud messaging, ChatConnect ensures reliable and synchronized communication. The app leverages RSA and AES hybrid encryption for secure message transfer, safeguarding user privacy and data integrity.

Key Features

  • Secure one-on-one messaging with real-time updates
  • RSA + AES hybrid encryption for private message exchange
  • Firebase Realtime Database for fast, synced communication
  • User authentication with Firebase Authentication
  • Photo sharing with cloud upload and retrieval
  • Temporary story posting and viewing
  • Clean and intuitive XML-based UI
  • Firebase Cloud Messaging for instant notifications

Technical Implementation

The application is built using Kotlin and adheres to Clean Architecture principles with a well-structured MVVM (Model-View-ViewModel) design pattern, ensuring separation of concerns, testability, and scalability.

ChatConnect is built using Android (Kotlin) with an XML-based UI and follows the MVVM architecture for separation of concerns. Firebase Authentication manages user sign-in, while Firebase Realtime Database handles message syncing in real-time. Media files like photos are stored in Firebase Storage.
