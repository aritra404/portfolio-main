Project Overview

MovieNow is a complete Android-based movie ticket booking system featuring two apps: a User App for customers to browse and book movie tickets, and an Admin App for cinema hall owners to manage their shows.

Users can explore available movies, select halls, choose show timings and seats, complete payments, and receive a PDF ticket with a unique ticket ID. On the other end, hall owners can log into the Admin App to list movies, set showtimes, update seat availability, and view bookings in real time.

The system ensures smooth coordination between audience and hall management with real-time updates powered by Firebase.

Key Features

  • Users can log in securely using OTP method
  • Browse currently running movies with details and posters
  • Select cinema hall, timing, and preferred seats
  • Make secure payments using Razorpay gateway
  • Get a downloadable PDF ticket with a unique ticket ID
  • Hall owners can log in securely using email/password
  • Admin App to add movies, configure show timings, and manage seats
  • View booking details for each show in real-time
  • Real-time data sync between both apps via Firebase Realtime Database
  • Clean and responsive XML-based UI in both apps

Technical Implementation

The system is built using Kotlin and XML-based UI, adhering to the MVVM architecture for clean and maintainable code. Firebase Authentication enables secure access for both users and hall admins. Firebase Realtime Database is used to sync data across the user and admin apps in real-time, ensuring bookings and updates reflect instantly.

Users complete payments securely through the Razorpay Payment Gateway, integrated into the User App for a smooth transaction experience.

After a successful booking, the User App generates a PDF ticket with a unique ID using Android's PdfDocument API. The Admin App allows cinema owners to manage their show listings and track user bookings seamlessly.
