Project Overview

SafeGuard is a smart Android safety app that provides voice-activated emergency support. When a user speaks a pre-set trigger word, the app records a 20-second audio clip, uploads it, and sends the audio link and real-time location to guardians via SMS. It then automatically initiates a call for direct help.

Powered by Speech Recognition, Firebase, and robust foreground services, SafeGuard offers fast, reliable emergency alerts—delivering peace of mind when it matters most.

Key Features

  • Voice-activated emergency detection
  • Customizable detection word
  • Automatic guardian alerts via SMS (with location & audio link)
  • Emergency call initiation to guardian
  • Real-time location sharing
  • Secure audio recording and Firebase Storage upload
  • Continuous background listening with Foreground Service
  • The app runs even when the screen is off

Technical Implementation

SafeGuard is built using Kotlin with an XML-based UI. It uses Android's Speech Recognition API to continuously listen for a predefined safe word while running as a foreground service, ensuring it remains active even when the app is in the background.

Upon detecting the safe word, the app records a 20-second audio clip, uploads it to Firebase Storage, and retrieves the real-time GPS location using the Fused Location Provider. These details are then sent via SMS to registered guardians using SmsManager, and a phone call is automatically initiated to ensure immediate attention.
